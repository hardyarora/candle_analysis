Execute the interactive feedback script to add feedback for engulfing candle patterns.

Run: `python3 scripts/add_feedback.py`

The script will interactively prompt for:
- Currency pair (e.g., NZD_CHF, GBP_USD)
- Timeframe (1D, 2D, 3D, or 4D)  
- Rating (1-10)
- Optional: Date, Pattern type, Notes

It analyzes the pattern, calculates metrics, and stores feedback automatically.
